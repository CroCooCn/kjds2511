# CRMEB 商城 Web 端启动指南

## 项目说明

这是 CRMEB 商城的 Web 端项目，基于 Vue 2 开发，实现了与小程序相同的功能。

## 功能特性

✅ **完整的商城功能**
- 首页展示
- 商品分类、搜索、详情
- 购物车管理
- 订单管理（下单、支付、退款）
- 用户中心（个人信息、地址、优惠券、收藏等）
- 营销活动（秒杀、拼团、砍价）
- 推广分销系统
- 资讯文章

✅ **移动端优化**
- 响应式设计，完美适配手机端
- 禁止缩放，提供原生 App 体验
- iOS Safari 全屏支持
- 流畅的触摸滚动

## 快速开始

### 1. 安装依赖

```bash
cd C:\Users\Lenovo\Desktop\跨境\kjds2511-main\web
npm install
```

### 2. 配置后端接口

修改 `vue.config.js` 中的代理配置：

```javascript
proxy: {
  '/api': {
    target: 'http://localhost:8080', // 修改为你的后端地址
    changeOrigin: true
  }
}
```

或者修改 `.env.development` 文件中的接口地址。

### 3. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:8081`

### 4. 手机访问（可选）

启动后，在同一局域网下，使用手机浏览器访问：

```
http://你的电脑IP:8081
```

查看电脑 IP 的方法：
```bash
ipconfig
```
查找 `IPv4 地址`

## 生产环境部署

### 1. 构建生产版本

```bash
npm run build:prod
```

构建完成后，会在 `dist` 目录生成静态文件。

### 2. 部署到服务器

将 `dist` 目录的内容上传到 Web 服务器（如 Nginx、Apache）。

### 3. Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    root /path/to/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理
    location /api {
        proxy_pass http://your-backend-server:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 主要页面路由

- `/` - 首页
- `/category` - 商品分类
- `/goods/:id` - 商品详情
- `/cart` - 购物车
- `/order/list` - 订单列表
- `/user` - 个人中心
- `/login` - 登录
- `/register` - 注册

更多路由请查看 `src/router/index.js`

## 常见问题

### 1. 端口被占用

修改 `vue.config.js` 中的 `port` 配置：

```javascript
const port = 8082 // 改为其他端口
```

### 2. API 请求失败

检查：
- 后端服务是否启动
- `vue.config.js` 中的代理配置是否正确
- 浏览器控制台的网络请求

### 3. 页面空白

检查：
- 浏览器控制台是否有错误
- 依赖是否正确安装
- Node.js 版本是否兼容（建议 Node 16-18）
